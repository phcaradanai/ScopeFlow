# ScopeFlow Thai - MVP Release Candidate (M8)

**เวอร์ชันเดิม: 0.1.0**

> หมายเหตุ: เอกสารนี้เป็น release note เก่าของ MVP RC ก่อน Guided Operating Mode V0.8 โปรดดู baseline ล่าสุดที่ `docs/RELEASE_NOTES_V0.8.md`

## แนะนำตัว (What is ScopeFlow Thai?)
ScopeFlow Thai เป็นแอปพลิเคชันจัดการเอกสารขอบเขตงาน (Scope) ที่ออกแบบมาสำหรับฟรีแลนซ์และทีมพัฒนาซอฟต์แวร์ไทยโดยเฉพาะ เพื่อช่วยป้องกันปัญหา Scope Creep ในการทำงาน โดยแอปนี้ทำงานแบบ Offline-First 100% รักษาความเป็นส่วนตัว ข้อมูลทั้งหมดจะถูกจัดเก็บเป็นไฟล์ธรรมดาในเครื่องของคุณเอง

## ฟีเจอร์หลัก (MVP Feature List)
- **File-First Workspace**: ข้อมูลเก็บเป็นไฟล์ Markdown (.md) และ YAML (.yaml) ในโฟลเดอร์บนเครื่องของคุณ สามารถสำรองข้อมูลและนำไปเปิดกับโปรแกรมอื่นได้
- **เอกสารครบวงจร**: รองรับ Quotation, Scope Baseline, Change Requests (CR), Defect Change Requests (DCR), Support Requests (SUP)
- **ระบบแบบฟอร์ม & คำนวณอัตโนมัติ**: มีฟอร์มสร้างใบเสนอราคา พร้อมระบบคำนวณ VAT และส่วนลด
- **ล็อกเอกสารและการอนุมัติ**: สามารถแนบหลักฐานการอนุมัติได้และ "ล็อก" เอกสารเพื่อป้องกันการแก้ไข หากจำเป็นต้องแก้ไขสามารถสร้าง Revision ใหม่ได้
- **Printable HTML Export**: สามารถสร้างชุดเอกสารสมบูรณ์ (Approval Pack) เป็น HTML เพื่อส่งให้ลูกค้า และสามารถสั่ง Print to PDF ได้อย่างสวยงามผ่านเบราว์เซอร์
- **Offline Backup & Restore**: สำรองข้อมูลโปรเจกต์เป็นไฟล์ ZIP ที่สมบูรณ์แบบได้ทันที พร้อมการบันทึกการตั้งค่า Company Profile ของคุณ

## คำสัญญาของเรา (Local / Offline / File-First Promise)
- ไฟล์ของคุณเป็นของคุณ เราจะไม่แอบส่งข้อมูลใดๆ ไปยังเซิร์ฟเวอร์ภายนอก
- ไม่มีโฆษณา ไม่มีการดักเก็บข้อมูลการใช้งาน (No Telemetry)
- ไม่จำเป็นต้องเชื่อมต่ออินเทอร์เน็ตเพื่อใช้งานแอปพลิเคชัน
- ไม่จำเป็นต้องสมัครสมาชิก ไม่บังคับล็อกอิน

## สิ่งที่แอปนี้ไม่มี (Non-Goals) และข้อจำกัดที่ควรทราบ
เพื่อความเรียบง่าย โปรดทราบว่า MVP RC เดิมยังไม่ใส่ฟีเจอร์ต่อไปนี้ลงในเวอร์ชันนั้น:
- **No Cloud Sync**: ไม่มีระบบซิงก์ข้อมูลขึ้นคลาวด์อัตโนมัติ (แต่คุณสามารถเอาโฟลเดอร์ Workspace ไปใส่ใน Google Drive หรือ Dropbox ได้เอง)
- **No Client Portal**: ไม่มีหน้าเว็บออนไลน์ให้ลูกค้ากดเข้ามาดูหรือกดยอมรับแบบออนไลน์ได้ (ให้ส่งไฟล์ Export แทน)
- **No Real e-Signature**: ไม่ใช่ระบบเซ็นเอกสารดิจิทัลแบบเข้ารหัสซับซ้อน เราใช้การแนบหลักฐานและบันทึกข้อมูลแทน
- **No Direct PDF Engine**: ไม่มีระบบสร้าง PDF โดยตรงในแอป (ให้สร้าง Approval Pack และใช้คำสั่ง Print ของคอมพิวเตอร์แทน)
- **No Billing/Payment tracking**: ไม่ใช่โปรแกรมทำบัญชีหรือออกใบกำกับภาษีเต็มรูปแบบ
- **Guided AI Assistance เพิ่มภายหลังใน V0.8**: baseline ใหม่รองรับ AI-assisted flow แบบ optional พร้อม fallback logic และ guard rails แล้ว
- แอปนี้ทำงานร่วมกับฐานข้อมูลแบบไฟล์เท่านั้น การแบ็คอัปจะเป็นรูปแบบไฟล์ ZIP

## วิธีการรายงานปัญหา (How to Report Feedback)
สำหรับผู้ทดสอบ Beta หากพบปัญหาหรือมีข้อเสนอแนะ สามารถกรอกข้อมูลตามรูปแบบที่เราเตรียมไว้ในไฟล์ `BETA_FEEDBACK_TEMPLATE.md` และส่งมาให้ทีมพัฒนาได้เลย ขอขอบคุณที่ร่วมเป็นส่วนหนึ่งในการทดสอบระบบ!
