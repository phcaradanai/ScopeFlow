# ScopeFlow Thai - Beta Feedback Form

ขอบคุณที่เข้าร่วมการทดสอบ ScopeFlow Thai ในช่วง Beta! เราอยากทราบความเห็นของคุณเพื่อให้แอปพลิเคชันตอบโจทย์ที่สุด กรุณาคัดลอกแบบฟอร์มนี้ เติมคำตอบ และส่งให้ทีมพัฒนาได้เลย

**1. อะไรเข้าใจง่าย? (What was easy to understand?)**
(ระบุฟีเจอร์หรือหน้าจอที่คุณคิดว่าใช้งานง่ายและทำความเข้าใจได้ไม่ยาก)
- 

**2. อะไรงง? (What was confusing?)**
(ระบุจุดที่คุณสับสน หรือต้องใช้เวลาทำความเข้าใจนาน)
- 

**3. ขั้นตอนไหนติดขัด? (Where did you get stuck?)**
(ระบุการทำงานที่รู้สึกว่ามันยาก หรือมีกระบวนการที่ซับซ้อนเกินไป)
- 

**4. เอกสารที่ Export ใช้ส่งลูกค้าได้ไหม? (Is the exported Approval Pack usable?)**
(ทดลองสั่ง Export ดูแล้ว หน้าตาเป็นอย่างไร ดูเป็นมืออาชีพและพร้อมใช้งานไหม?)
- 

**5. Quotation Form พอใช้งานจริงไหม? (Is the quotation form practical?)**
(ลองกรอกฟอร์มใบเสนอราคาดูแล้ว ระบบคำนวณถูกไหม ฟิลด์ที่ให้มากพอไหม?)
- 

**6. Approval / Lock / Revision เข้าใจไหม? (Are approval and revision workflows clear?)**
(คุณเข้าใจระบบการล็อกไฟล์เมื่ออนุมัติและบังคับให้สร้าง Revision ไหม? ใช้งานยากไปไหม?)
- 

**7. Backup / Restore ใช้ง่ายไหม? (Is backup/restore easy to use?)**
(ทดสอบสำรองข้อมูลเป็น ZIP และดึงกลับมาใช้งาน คุณมีปัญหาอะไรไหม?)
- 

**8. Feature ไหนจำเป็นที่สุดถัดไป? (What is the most needed next feature?)**
(คุณคิดว่าแอปพลิเคชันควรจะมีอะไรเพิ่มเป็นลำดับถัดไป? **หมายเหตุ:** โปรดทราบว่าเราไม่ทำระบบคลาวด์ หรือระบบล็อกอิน)
- 

**9. Bug / Error ที่เจอ (Any bugs or errors?)**
(เจอจอขาว หรือเกิด Error อะไรขึ้นบ้าง? โปรดเล่ารายละเอียดให้เราฟัง)
- 

**10. ข้อเสนอแนะอื่น ๆ (Any other suggestions?)**
(สิ่งที่อยากบอกเพิ่มเติม)
